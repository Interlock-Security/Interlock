# Build Webinix 2 - Windows

- **Build Tools for Microsoft Visual Studio**

```sh
git clone https://github.com/alifcommunity/webinix.git
cd webinix\build\Windows\MSVC
nmake
```

- **MinGW**
```sh
git clone https://github.com/alifcommunity/webinix.git
cd webinix\build\Windows\GCC
mingw32-make
```

- **TCC**
```sh
git clone https://github.com/alifcommunity/webinix.git
cd webinix\build\Windows\TCC
mingw32-make
```
