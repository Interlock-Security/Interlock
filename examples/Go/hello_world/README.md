# Webinix Examples - Golang

### Windows
```sh
git clone https://github.com/alifcommunity/webinix.git
cd webinix\examples\Go\hello_world
go clean
go build -o hello_world.exe
hello_world
```

### Linux
```sh
git clone https://github.com/alifcommunity/webinix.git
cd webinix/examples/Go/hello_world
go clean
go build -o hello_world
./hello_world
```

### macOS
